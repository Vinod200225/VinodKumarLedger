Add real PNG icons here:

  192.png  → 192x192 px
  512.png  → 512x512 px

Until then, the PWA install banner won't show on iOS/Android.

Quick way to generate: use https://realfavicongenerator.net or any image
editor. Theme color is #6366f1, background #0f172a.

Until you add real PNGs, the included icon.svg is used as a fallback
in index.html via <link rel="icon">.
